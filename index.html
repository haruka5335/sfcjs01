HTMLにおいて使用出来るタグを調べた。
今回は情報基礎1で利用した基本的なタグを中心にまとめていくこととする。

1.<p>、</p>…HTML内において段落を作る際に使用する。


       <p>こんな風にね</p>

2.<br>…改行。
例えば、こんな風に少し長めの文章だったりなんだりするとなかなか改行されず画面いっぱいに文字が広がり、改行されたとしても訳の分からないところで改行されてふざけんな読みにくいんだよボケって思うじゃろ？
でもこの<br>を使うと超簡単！
改行したいとことに打ち込むだけで、こんなに読みやすい文章になったよ！
sugoi！私<br>と<p>、</p>大好き！


       こんな風にね.<br>

3.<img>…画像挿入。
画像を使いたい時に使うよ。


       <img src=屋上.jpg alt=屋上 >
実際入れるとこんな風になるよ

4.<input>…入力エリアの作成。
何か入力させたいときに使うよ。だいたいボタンとセットで使うよ。


           <input type=text>
           <input type=submit value="送信">
実際に使うとこうなるよ
　　

以降はCSSにおける属性についての記述である。
基本的なものをまとめた(つもり)であるので、分からなくなったら参考程度に見るように。

1.文字の色を変える

文字の色を変えたいときに使うよ。

       CSS: h4.ex1{color: red;
            }
       HTML: <h4 class="ex1">こんな風にね></h4>

こんな風にね

2.文字の大きさを変える

文字の大きさを変えたいときに使うよ。

               CSS: h4.ex2{font-size: 30px;
               }
               HTML: <h4 class="ex2">こんな風にね></h4>

こんな風にね

指定の仕方

ここで使っているのはクラス指定。
まずHTMLでクラスを指定します。

       HTML: <h4 class="ex2">こんな風にね></h4>
の、「h4 class="ex2"」の部分が指定。
「h4の、クラスがex2の所」を変更するようにCSSに書くときは、こう。

       CSS: h4.ex2{
            }
頑張ってね。
